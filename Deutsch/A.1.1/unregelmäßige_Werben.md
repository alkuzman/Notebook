# Irregular Verbs

|          |           | lesen     | essen    |
| -------- | --------- | --------- | -------- |
| Singular | ich       | lese      | esse     |
|          | du        | **liest** | **isst** |
|          | er/sie/es | **liest** | **isst** |
| Plural   | wir       | lesen     | essen    |
|          | ihr       | lest      | **isst** |
|          | sie       | lesen     | essen    |
| formell  | Sie       | lesen     | essen    |
